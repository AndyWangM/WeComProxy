💡 使用说明

API调用示例:

curl -X POST http://localhost:3000/webhook/alert_app \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{"sender": "监控系统", "time": "2024-03-24 15:00:00", "content": "服务器状态异常"}'
⚠️ 安全提醒:
  • Token是访问API的唯一凭证，请妥善保管
  • 不要在公共场所或不安全的环境中暴露Token
  • 如果Token泄露，请立即重新生成新Token
  • 建议定期更换Token以提高安全性
